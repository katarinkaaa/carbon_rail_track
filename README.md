# carbon_rail_track
Проект по расчету выбросов CO2 на жд транспорте

## Для запуска кода 
откройте VisualStudioCode, папка railway. В терминале в этой же папке прописываем код для запуска: python -m uvicorn backend.main:app --reload

# Обзор фалов

## backend/main.py
Язык/стек: Python + FastAPI
Назначение:
— Серверное API для расчёта экологической экономии.
— Принимает POST-запрос /api/calculate с данными (маршрут, масса, тип контейнера).
— Вычисляет:

расстояние по координатам (через haversine),
выбросы CO₂ для ж/д и условного «судна» (ship_plus или ship),
экономию (saved_kg) и сколько деревьев это «равносильно».
— Отдаёт JSON: { rail_kg, ship_kg, saved_kg, trees, coords, ... }
— Также отдаёт список маршрутов (/api/routes) и обслуживает статику (frontend).

## backend/data/routes_with_coords.py
Язык: Python
Назначение:
— Хранит статические данные проекта:

ROUTES: { "1": { "name": "...", "stations": ["Достык", "Алматы", ...] } }
COORDINATES: { "Достык": [45.123, 80.456], ... } — широта/долгота.
haversine(lat1, lon1, lat2, lon2): функция расчёта расстояния между двумя точками на Земле (в км).

## frontend/index.html
Язык/стек:

HTML5 — структура,
Tailwind CSS (через CDN) — стили,
Vue 3 (Composition API) — логика интерфейса,
Leaflet.js — интерактивная карта
Назначение:
— Единая веб-страница-калькулятор.
— Позволяет выбрать: маршрут, массу, тип контейнера → нажать «Рассчитать».
— Отправляет запрос на /api/calculate.
— Отображает:

карту с построенным маршрутом,
экономию CO₂ и количество «посаженных деревьев»
